# super-temp-config-generator
script to generate a SUPER temp config plist with customizable parameters

IMPORTANT 
Set Jamf Pro Parameters 4,5,6, and 7 to the following:
Parameter 4: Temporary Config Name, this is the name of the temporary config file in the SUPER 'configs' folder
Parameter 5: allow macOS upgrades | true or false
Parameter 6: Targeted macOS version | minor or major target
Parameter 7: Verbose mode to show more information in logs | 1 - Enabled 0 - Disabled
